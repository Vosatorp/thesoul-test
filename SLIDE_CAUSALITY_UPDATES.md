# Slide Updates: Causality Caveats

> Оговорки и уточнения для включения в презентацию TheSoul Group.

## Слайд: Optimal Duration (8-10 мин)

### Текущая формулировка (корректировка):

Вместо: «8-10 мин видео генерируют наибольший revenue»

Использовать: «8-10 мин видео **ассоциируются** с более высоким revenue, однако этот вывод требует осторожной интерпретации»

### Footnote / мелкий текст:

- Выборка: только 104 видео из ~11K (<1%)
- 34% сконцентрированы в 5 каналах — возможен channel confounding
- Within-channel: 8-10 мин лучше в 13/34 каналах (38%)
- Для подтверждения нужен A/B тест на одном канале

## Слайд: Cross-Platform Strategy

### Текущая формулировка (корректировка):

Вместо: «Кросс-постинг увеличивает reach/revenue»

Использовать: «Видео, размещённые на нескольких платформах, **показывают более высокие метрики** — но это может отражать selection bias (для кросс-постинга выбирают лучший контент)»

### Footnote / мелкий текст:

- Кросс-постнутые видео имеют в 3.2x больше просмотров → возможен selection bias
- Within-channel: кросс-пост лучше по views в 21/57 каналах (37%)
- Within-channel: кросс-пост лучше по revenue в 28/57 каналах (49%)
- Направление: корреляция, не causation. Нужен контролируемый эксперимент.

## Общий caveat для всей презентации

Все выводы основаны на наблюдательных данных (observational data). Корреляции не равны причинно-следственным связям. Для подтверждения рекомендуется:
1. **A/B тесты** — случайно назначать формат/длительность и замерять эффект
2. **Propensity score matching** — подобрать «похожие» видео разных длительностей для fair comparison
3. **Difference-in-differences** — сравнить каналы до и после смены стратегии
